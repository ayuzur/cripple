# cripple

c ripple

a terminal screen saver that replicates the ripples caused by rainfall

## build

compile the main file. Link the math library if necessary

gcc
`gcc main.c -lm -o cripple`

cl
`cl /Fe:cripple main.c`

## usage

run the executable

ctrl+c to stop
